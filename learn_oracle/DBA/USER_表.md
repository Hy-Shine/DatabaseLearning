## 用户表

```plsql
--查看当前用户下所有表的字段注释
SELECT * FROM USER_COL_COMMENTS ORDER BY TABLE_NAME
TABLE_NAME,COLUMN_NAME,COMMENTS
```
### 表与字段

```plsql
---查看当前用户下的所有表(TANLE_NAME,TABLESPACE_NAME,STATUS)
select * from user_tables;
---查看所有用户下的表,包括系统用户(OWNER,TANLE_NAME,TABLESPACE_NAME,STATUS)
select * from all_tables;
---查看当前用户下表字段(TABLE_NAME,COLUMN_NAME,DATA_TYPE,DATA_LENGTH)
SELECT * FROM USER_TAB_COLUMNS;
---查看所有用户下表字段(OWNER,TABLE_NAME,COLUMN_NAME,DATA_TYPE,DATA_LENGTH)
SELECT * FROM ALL_TAB_COLUMNS;
```

### 注释

```plsql
--- 查看当前用户下的表注释(TABLE_NAME,TABLE_TYPE,COMMENTS)
SELECT * FROM USER_TAB_COMMENTS;
--- 查看所有用户下的表注释(OWNER,TABLE_NAME,TABLE_TYPE,COMMENTS)
SELECT * FROM ALL_TAB_COMMENTS;
--- 查看当前用户下表字段注释(TABLE_NAME,COLUMN_NAME,COMMENTS)
SELECT * FROM USER_COL_COMMENTS;
--- 查看所有用户下表字段注释(OWNER,TABLE_NAME,COLUMN_NAME,COMMENTS)
SELECT * FROM ALL_COL_COMMENTS;
```
