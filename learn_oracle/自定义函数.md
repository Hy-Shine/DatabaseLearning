## 自定义函数

```plsql
---自定义函数语法格式
create or replace function 函数名(参数1 模式 参数类型,参数2 模式 参数类型)
return 返回值类型
as
  变量1 变量类型;
  变量2 变量类型;
begin
    函数体;
end 函数名;
```

- 模式类型

  ```plsql
  ---参数的模式有3种:(如果没有注明, 参数默认的类型为in)
  in: 为只读模式, 在函数中, 参数的值只能被引用, 不能被改变;
  out: 为只写模式, 只能被赋值, 不能被引用;
  in out: 可读可写
  ```
