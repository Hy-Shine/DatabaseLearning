## 常见数据类型

### 字符类型
```plsql
VARCHAR2(length)：存储可变的长度的字符串，length:是字符串的最大长度，默认不填时为1，最大长度不超过4000字节。
CHAR(length)：存储固定长度的字符串，length:字符串的固定长度大小，默认不填为1，最大长度不超过2000字节。
```
### 日期类型
```plsql
DATE：时间类型，存储的是日期和时间，包括年、月、日、时、分、秒。
TIMESTAMP：时间类型：存储的不仅是日期和时间，还包含了时区。例：内置函数systimestamp获取的就是timestamp类型 
```
### LOB类型
```plsql
CLOB：大字段类型，存储的是大的文本，比如：非结构化的txt文本，字段大于4000长度的字符串。
BLOB：二进制类型：存储的是二进制对象，比如图片、视频、声音等转换过来的二进制对象。
BCLOB：
NFILE：二进制文件，存储在数据库外的系统文件，只读的，数据库会将该文件当二进制文件处理
```
### 数值类型
```plsql
--NUMBER
NUMBER(a,b)：数值类型：存储数值类型，可以存整数，也可以存浮点型。
a代表数值的最大位数：整数位+.+小数位，b为小数位。
例：
number(6,2)，输入123.123，实际存入：123.12 。
number(4,2)，输入12312.345，实际存如：提示不能存入，超过存储的指定的精度。
--INTEGER
等价于number(38,0)
```
